When to use which?

At the reference size both modes are identical. Mode A keeps the rig proportional to the sofa — the closest match to the original look at any size. Mode B treats the lights as fixed physical fixtures, so big soft sources stay relatively soft as the sofa grows.

Reference rig dimensions (advanced)

The sofa size the rig is tuned for right now. When the input equals the reference, the output matches the source T3D 1:1.

How axes & scaling work

Axes: world X↔width(453), Y↔depth(274), Z↔height(77) — inferred from the fill (SourceWidth 500 ≈ 453). Positions scale per-axis; AttenuationRadius follows the distance; rotations stay fixed.

Calculation formula
1. Scale factors (reference rig 453 × 274 × 77):
sX = W / 453    sY = D / 274    sZ = H / 77
rotated 90°:  sX = D / 453,  sY = W / 274
2. New light position:
X′ = X · sX    Y′ = Y · sY    Z′ = Z · sZ
3. Change in distance to the sofa (per light):
d0 = √(X² + Y² + Z²)    d1 = √(X′² + Y′² + Z′²)    k = d1 / d0
4. Effective source radius:
spot:  R = SourceRadius    rect:  R = √(SourceWidth · SourceHeight / π)
5. Mode A — source sizes scale (× k):
Intensity′ = Intensity · k²    SourceWidth/Height/Radius/BarnDoorLength · k
5. Mode B — source sizes fixed:
Intensity′ = Intensity · (k²·d0² + R²) / (d0² + R²)  ≈  Intensity · kp
p = 2·d0² / (d0² + R²)    sharp key → p ≈ 2,  soft fill → p ≈ 1.7
6. Attenuation & unchanged params:
AttenuationRadius′ = AttenuationRadius · k
rotations, color, temperature — unchanged

Why not a "pure" inverse square: for large soft sources (fill, left_rim) R is comparable to the distance — the near field — so illuminance falls off gentler than k². When the input equals the reference, k = 1 → the output matches the source rig 1:1.
