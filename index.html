Расчёт по источникам

ИсточниктипXYZkИнтенс.p

Формула расчёта

1. Коэффициенты масштаба (эталон рига 453 × 274 × 77):
sX = W / 453    sY = D / 274    sZ = H / 77
поворот 90°:  sX = D / 453,  sY = W / 274
2. Новая позиция источника:
X′ = X · sX    Y′ = Y · sY    Z′ = Z · sZ
3. Изменение дистанции до дивана (своё для каждого света):
d0 = √(X² + Y² + Z²)    d1 = √(X′² + Y′² + Z′²)    k = d1 / d0
4. Эффективный радиус источника:
spot:  R = SourceRadius    rect:  R = √(SourceWidth · SourceHeight / π)
5. Режим A — размеры источников масштабируются (× k):
Intensity′ = Intensity · k²    SourceWidth/Height/Radius/BarnDoorLength · k
5. Режим B — размеры источников неизменны:
Intensity′ = Intensity · (k²·d0² + R²) / (d0² + R²)  ≈  Intensity · kp
p = 2·d0² / (d0² + R²)    резкий key → p ≈ 2,  мягкая заливка → p ≈ 1.7
6. Затухание и неизменные параметры:
AttenuationRadius′ = AttenuationRadius · k
повороты, цвет, температура — без изменений

Почему не «чистый» обратный квадрат: у больших мягких источников (fill, left_rim) R сравнимо с дистанцией — это ближняя зона, поэтому падение освещённости мягче k². При входных габаритах = эталонным k = 1 → вывод совпадает с исходным ригом 1:1.
